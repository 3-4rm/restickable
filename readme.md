# Restickable.js
# react-restickable.js

Makes html elements stick/unstick based on vertical scroll positon.
**react-restickable** is a wrapper to use with react.js.

This is a fork of [sticky-kit] with removed dependancy on jQuery and has some extra features.

#### [Demo and usage]


**To run demo locally:**


```sh
$ git clone https://github.com/3-4rm/restickable.git
```
Then cd into restickable/demo
```sh
$ npm install
```
Then
```sh
$ webpack --watch
```

Then open .../restickable/demo/index.html or .../restickable/demo/as-script.js as local file in your browser.
